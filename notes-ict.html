Definition: The Internet is a global network of networks that allows computers to communicate using the TCP/IP protocol suite.

Core concepts

  • IP Address: Unique address for devices on the internet.
  • DNS: Domain Name System translates names to IPs.
  • HTTP/HTTPS: Protocols for loading web pages; HTTPS uses encryption (TLS).

Short exercise

  1. Describe how DNS resolves www.example.com to an IP address.
  2. List three differences between HTTP and HTTPS.
  3. Give two examples of URL components (protocol, hostname, path).
